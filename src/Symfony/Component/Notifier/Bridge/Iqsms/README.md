Iqsms Notifier
==============

Provides [Iqsms](https://iqsms.ru) integration for Symfony Notifier.

DSN example
-----------

```
IQSMS_DSN=iqsms://LOGIN:PASSWORD@default?from=FROM
```

where:
 - `LOGIN` is your IQSMS login
 - `PASSWORD` is your IQSMS password
 - `FROM` is the sender
