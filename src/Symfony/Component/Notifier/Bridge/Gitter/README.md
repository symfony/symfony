Gitter Notifier
===============

Provides [Gitter](https://gitter.im) integration for Symfony Notifier.

DSN example
-----------

```
GITTER_DSN=gitter://TOKEN@default?room_id=ROOM_ID
```

where:
 - `TOKEN` is your Gitter token
 - `ROOM_ID` is your Gitter room id
