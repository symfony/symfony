Infobip Notifier
================

Provides Infobip integration for Symfony Notifier.

DSN example
-----------

```
INFOBIP_DSN=infobip://AUTH_TOKEN@INFOBIP_HOST?from=FROM
```

where:
 - `AUTH_TOKEN` is your Infobip auth token
 - `INFOBIP_HOST` is your Infobip host
 - `FROM` is the sender
