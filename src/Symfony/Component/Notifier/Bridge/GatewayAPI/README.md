GatewayAPI Notifier
===============

Provides GatewayAPI integration for Symfony Notifier.

DSN example
-----------

```
// .env file
GATEWAYAPI_DSN=gatewaypi://TOKEN@default?from=FROM
```

where:
 - `TOKEN` is API Token (OAuth)
 - `FROM` is sender name

See your account info at https://gatewayapi.com
