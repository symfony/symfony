LINE Bot Bridge
=============

Provides [LINE Bot (Messaging API)](https://developers.line.biz/en/docs/messaging-api/sending-messages/) integration for Symfony Notifier.

DSN example
-----------

```
LINE_BOT_DSN=linenotify://TOKEN@default
```
