Beanstalkd Messenger
====================

Provides Beanstalkd integration for Symfony Messenger.

Full DSN with options: `beanstalkd://<ip>:<port>?tube_name=<name>&timeout=<timeoutInSeconds>&ttr=<ttrInSeconds>`
