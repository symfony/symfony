Twilio Notifier
===============

Provides [Twilio](https://www.twilio.com) integration for Symfony Notifier.

DSN example
-----------

```
TWILIO_DSN=twilio://SID:TOKEN@default?from=FROM
```

where:
 - `SID` is your Twillio ID
 - `TOKEN` is your Twilio token
 - `FROM` is your sender
