Amazon Notifier
===============

Provides [Amazon SNS](https://aws.amazon.com/de/sns/) integration for Symfony Notifier.

DSN example
-----------

```
AMAZON_SNS_DSN=sns://ACCESS_ID:ACCESS_KEY@default?region=REGION
```
